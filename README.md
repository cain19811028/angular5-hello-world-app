# angular5-hello-world-app
測試 angular5 快速建立簡易 app

### Step 1：安裝 Node.js
* 確認 node -v 有顯示 Node.js 版本
* 確認 npm -v 有顯示 npm 版本

### Step 2：安裝 Angular CLI
```
npm install -g @angular/cli
```

### Step 3：建立一個新專案
```
ng new angular5-hello-world-app
```

### Step 4：啟動服務
```
cd angular5-hello-world-app
ng serve --open
```
